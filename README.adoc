=== Installation

==== After booted into the CachyOS ISO
. Launch the installer.
. *Bootloader*: systemd-boot
. *Partitions*: choose 3rd M2 / erase disk / btrfs
. *Desktop*: Plasma Desktop
. Check https://wiki.cachyos.org/configuration/post_install_setup/[CachyOS' official page]

=== Firewall config (example AusweisApp)
* create ufw config file for application: +
`/etc/ufw/applications.d/ufw-ausweisapp`
----
[Ausweisapp]
title=Ausweisapp
description=Ausweisapp2
ports=24724
----
* enable App: +
`sudo ufw allow ausweisapp`
